# Librarian-Demo
- 스프링 부트 및 관련 기술 연습용 Backend 데모
  - 멀티모듈로 구성

    - application : 트랜잭선 단위로 기능을 수행
    - domain-*** : 해당 도메인 관련 엔티티, 서비스, 레포지토리 인터페이스. 핵심적인 비즈니스 로직도 수행.
    - infrastructure : 데이터베이스 등 기술 지원
  
  - 향후 과제
